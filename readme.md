# Hollacabs - Ride Matching Service
Match ride requests to the nearest available driver. Designed for maximal efficiency to handle heavy load. 

Tech Stack:
- Node
- Koa
- Cassandra
- Redis
- Docker Compose
- AWS ELB, ECS, EC2, SQS
- Jest
